# Software gestionale per un negozio vegano

## Descrizione
Questo progetto, prodotto durante il Master in Data Science di Profession AI, consiste nel realizzare un software gestionale per un negozio vegano utilizzando il linguaggio **Python**. 
L'obiettivo principale è fornire uno strumento che consenta di gestire l'inventario, tracciare gli ordini e ottenere report dettagliati sulle vendite. 
Il software è testuale, quindi utilizzabile da riga di comando.

Il progetto è stato sviluppato all'interno di un notebook Jupyter in Google Colab e utilizza librerie Python per la gestione e la visualizzazione dei dati.

## Funzionalità
- Registrare nuovi prodotti, con nome, quantità, prezzo di vendita e prezzo di acquisto.
- Elencare tutti i prodotti presenti.
- Registrare le vendite effettuate.
- Mostrare i profitti lordi e netti.
- Mostrare un menu di aiuto con tutti i comandi disponibili.

## Comandi
- **aiuto**: mostra e descrive i comandi che è possibile utilizzare.
- **aggiungi**: inserisce il prodotto o i prodotti in magazzino.
- **elenca**: elenca tutti i prodotti presenti in magazzino.
- **vendita**: consente di vendere prodotti.
- **profitti**: mostra gli utili lordi e netti delle vendite totali.
- **chiudi**: chiude il programma.

## Tecnologie utilizzate
- **Linguaggio**: Python
- **Ambiente di sviluppo**: Jupyter Notebook

## Come eseguire il software
1. Clona o scarica il repository.
2. Apri il file `software_gestionale_negozio_vegano.ipynb` con Jupyter Notebook o Google Colab.
3. Segui le istruzioni presenti nel notebook per eseguire il software.
